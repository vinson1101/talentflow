# product_manager_batch_001

本批用于校准产品经理岗位的 shortlist / 建联决策。

## 重点观察

- 产品 adjacent 候选人是否被错误抬高
- 高 willingness、低 recruitability 是否被误推
- recoverable 类型是否被正确落到 `maybe / C`
- 项目经理 / 方案型 / 实施型背景是否被误吸成产品

## 边界样本

- `王宏宇`：项目经理背景，偏交付，验证低 `match_fit` / `hard_mismatch`
- `许罗栋`：解决方案工程师背景，验证非产品误推
- `滕锋帅`：软件工程师 / 实施背景，验证非产品误推
- `张金莎`：年限不足，验证低年限样本不要被抬高
- `杨淑婷`：工作年限不足，验证不应推进
